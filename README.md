# React Realtime Todo List

A polished React todo list app with instant updates, search, filters, priority labels, due dates, editing, completion tracking, and localStorage persistence.

## Features

- Add todos with priority and optional due date
- Mark todos complete or active in real time
- Edit todo text, priority, and due date
- Delete single todos or clear completed todos
- Filter by all, active, completed, and overdue
- Search todos instantly
- Live stats for total, active, completed, and overdue tasks
- Saves automatically in the browser with localStorage

## Run Locally

This project is dependency-free and uses React from a CDN, so you can run it without installing packages.

Open `index.html` in your browser, or serve it locally:

```bash
npm start
```

Then visit:

```text
http://localhost:5173
```
